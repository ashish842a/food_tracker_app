
# Food Tracker App

A simple and efficient application to track your daily food intake and nutritional information.

## Features

- Log meals and snacks throughout the day
- Track calories and macronutrients
- View nutritional summaries
- Maintain a food diary
- Set daily intake goals

## Installation

```bash
git clone https://github.com/ashish842a/food_tracker_app.git
cd food_tracker_app
npm install
```

## Usage

1. Start the development server: `npm run dev`
2. Add meals with food items and portions
3. View your daily nutrition summary
4. Monitor progress toward your goals

## Tech Stack

- Node.js (Backend)
- React with Vite (Frontend)

## Requirements

- Node.js 14+
- npm or yarn
- Dependencies listed in `package.json`
